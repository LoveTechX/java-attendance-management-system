# Java Attendance Management System

## Project Overview
This is a console-based Java mini project that manages student attendance for a specific class.

## Features
- Student Registration
- Day-wise Attendance Marking
- Attendance Record Viewing
- Attendance Percentage Calculation
- Shortage Identification (Below 75%)
- Menu-driven Program

## Concepts Used
- Variables & Data Types
- String Class
- if / if-else
- for loop
- do-while loop
- break and continue
- 1D Arrays
- 2D Arrays

## How It Works
Attendance is stored using a 2D array:
attendance[student][day]

Each row represents a student and each column represents a working day.
